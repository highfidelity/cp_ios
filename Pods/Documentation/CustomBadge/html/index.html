    CustomBadge 2.0 Reference

    Description: With this class you can draw a typical iOS badge indicator with a custom text on any view. Please use the allocator customBadgeWithString to create a new badge. In this version you can modfiy the color inside the badge (insetColor), the color of the frame (frameColor), the color of the text and you can tell the class if you want a frame around the badge.
